# 🧠 TypeScript Daily Challenge

> **Golden rule**: no AI to write the functions. Brain first.

## 🎯 Goal

2 TypeScript functions per day, pushed to GitHub every single day, no exception.

This challenge is about keeping your reflexes sharp and thinking for yourself especially since AI is everywhere in our projects. The goal isn't to avoid AI entirely, it's to make sure _you_ stay the one who actually understands the code.

## 📁 Structure

```
src/
├── week-01/
│   ├── day-01.ts        ← you code here
│   ├── day-01.test.ts   ← don't touch this
│   ├── day-02.ts
│   └── ...
├── week-02/
└── ...
```

## 🚀 Quick start

```bash
npm install

# Run tests once
npm test

# Watch mode — tests re-run on every save
npm run test:watch
```

## 📋 Rules

1. **Code** only in `day-XX.ts`
2. **Don't modify** `.test.ts` files
3. **Commit** every day with a clear message: `day-01: sumArray + factorial ✅`
4. **No AI** to write the functions — that's the whole point
5. **30 min max** per function. If you're stuck, grab a pen and paper
